问题说明

问题原因： index.js中的loadAssets()函数覆盖了assets.js中的同名函数，导致批量导入功能无法正常显示。

修复方案： 将index.js中的函数重命名为loadAssetsPage()，避免函数名冲突。

测试步骤

  1. 点击下面的按钮打开修复后的主页面
  2. 点击左侧菜单的"设备台账"
  3. 查看是否显示批量导入按钮
  4. 测试批量导入功能
